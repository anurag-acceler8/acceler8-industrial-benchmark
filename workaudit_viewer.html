Employee work audit

Load your JSON by path (served over HTTP) or pick a file. Cards show role, risk, productivity, and a snapshot of work patterns.

Same-folder default works if you run e.g. cd uploads && python -m http.server 8765 then open http://localhost:8765/workaudit_viewer.html. For another location, set the path relative to that server root.

Viewer only — data stays in your browser.
